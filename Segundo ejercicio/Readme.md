Algoritmo encargado de imprimir una serie de numeros de forma ordenada:

Ejemplos:

▶ Entrada
(1, 3, 2, 0, 7, 8, 1, 3, 0, 6, 7, 1)→ salida: 123 1378 167

▶ Caso con bloque vacío
(2, 1, 0, 0, 3, 4) → salida: 12 X 34


Adicionalmente, se imprime salidas si tenemos 0 en los bordes del arreglo:

▶ Ejemplo
(0, 1, 3, 2, 0, 0, 7, 8, 1, 3, 0, 6, 7, 1, 0)→ salida: 123 X 1378 167

Si tenemos dos o más ceros en los bordes.

▶ Ejemplo
(0, 0, 1, 3, 2, 0, 0, 7, 8, 1, 3, 0, 6, 7, 1, 0, 0, 0)→ salida: X 123 X 1378 167 X X
